# Secure Login System

A secure login system developed using Flask and SQLite that provides user registration, login, password hashing with bcrypt, session management, and optional two-factor authentication (2FA).

## Features

- User Registration
- Secure Login
- Password Hashing using bcrypt
- SQLite Database
- Session Management
- Optional Two-Factor Authentication (2FA)
- Basic Input Validation

## Technologies Used

- Python
- Flask
- SQLite
- bcrypt
- pyotp
- qrcode
